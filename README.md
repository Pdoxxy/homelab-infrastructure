# Homelab Infrastructure

This repository documents my home network, Ubuntu server and self-hosted
services.

The environment is used to develop practical skills in networking, Linux
administration, containerization, security and infrastructure documentation.

## Current environment

### Network

- Bahnhof fiber connection
- UniFi Cloud Gateway Ultra
- UniFi U7 Pro wireless access point
- UniFi Switch Lite 8 PoE
- UniFi Flex Mini 2.5G
- Public IPv4 and IPv6 connectivity

### Server

- Ubuntu Server
- Docker Compose
- Home Assistant
- Pi-hole
- Plex Media Server
- VPN-routed media services

## Purpose of this repo

- Document the physical and logical network architecture
- Introduce VLAN-based network segmentation
- Apply least-privilege firewall policies
- Manage services through Docker Compose configurations
- Improve monitoring, backup and recovery procedures

## Security notice

This repository contains sanitized examples rather than direct exports from
the environment. Credentials, keys, public addresses and other
sensitive identifiers are excluded.
